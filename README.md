# 🧮 Calculadora Simples

Uma calculadora básica desenvolvida com **HTML, CSS e JavaScript**, criada com o objetivo de praticar lógica de programação e manipulação do DOM.

---

## 🚀 Funcionalidades

- ➕ Adição  
- ➖ Subtração  
- ✖️ Multiplicação  
- ➗ Divisão  
- 🧹 Limpar valores  

---

## 🎯 Objetivo

Este projeto foi desenvolvido como prática para:

- Estruturação com HTML  
- Estilização com CSS  
- Interatividade com JavaScript  
- Manipulação de elementos via DOM  

---

## 🖥️ Preview

Interface simples, direta e funcional.

---
## ⚙️ Como usar

1. Abra o arquivo `calculadora.html` no navegador  
2. Insira dois valores  
3. Escolha a operação  
4. Clique em **Calcular**  

---

## 📌 Observação

Este é um projeto simples, feito para aprendizado e evolução prática.  
Não possui validações avançadas ou tratamento de erros complexos.

---

## 🧠 Aprendizado

Durante esse projeto, foram trabalhados conceitos como:

- Entrada de dados pelo usuário  
- Eventos em JavaScript  
- Operações matemáticas básicas  
- Organização de código
